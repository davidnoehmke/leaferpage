# LEAFerservice Shopify Theme

Shopify Online Store 2.0 theme for LEAFerservice.

## Included

- Dynamic product descriptions and Shopify metafields
- Recipe and ingredient presentation
- Substrate configurator
- Product cross-selling
- Collection-specific SEO content and FAQs
- Open Graph and structured product data
- Responsive product, collection, search and cart layouts
- German and English storefront copy

## Shopify metafields

The product template reads these optional metafields:

- `leafer.mixing_ratio`
- `leafer.ingredients`
- `leafer.cross_selling`
- `leafer.short_claim`

## Validation

The complete theme was validated with Shopify Theme Check before transfer.
